# neuro_nets
## Создание проекта
```poetry new poetry-demo```
## Cделать виртуальное окружение (зайти в папку)
`poetry shell`
## Выбрать интерпретатор 
`ctrl+shift+p -> Выбрать интерпретатор`
скопировать путь из терминала. Вставить путь. Перейти в папку scripts. Выбрать python.
## Установка пакетов
`poetry add <имя пакета1> <имя пакета2>...`
## Как делать commit
`git add .`
`git commit -m "сообщение"`
`git push`
